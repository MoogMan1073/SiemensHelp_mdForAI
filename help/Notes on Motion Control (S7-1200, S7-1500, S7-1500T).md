## Motion Control (S7-1200)

### Automatically apply drive or encoder values during configuration (offline)

Not all configuration parameters are automatically applied when connecting a SINAMICS V90 PN via HSP. Deactivate the "Automatically apply drive values during configuration (offline)" option and apply the configuration parameters of the drive and encoder manually in the configuration of the technology object.
