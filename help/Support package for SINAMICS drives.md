## Overview of SINAMICS HSPs

### Overview of the online helps for SINAMICS HSPs

This section contains the online helps for the hardware support packages (HSPs) of SINAMICS drives.

The individual HSPs are each assigned to a firmware version of SINAMICS.

The following helps are supplied for each drive:

- User interface help for the function view
- Online help for the drive parameters
- Online help for the drive alarms
