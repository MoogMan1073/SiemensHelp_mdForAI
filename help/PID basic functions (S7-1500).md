## Differences compared to CONT_C S7-300/400 (S7-1500)

The CONT_C instruction is created with optimized block access.

All parameters are retentive. The retentivity cannot be changed. The actual values of retentive tags are only updated when you download CONT_C completely.

[Downloading technology objects to device](Configuring%20a%20software%20controller.md#downloading-technology-objects-to-device)

Otherwise, the behavior of CONT_C on an S7-1500 CPU is exactly the same as on an S7-300, S7-400 CPU.
