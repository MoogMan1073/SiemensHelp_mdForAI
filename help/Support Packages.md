# Support Packages

## Subsequent provision of contents via support packages

By installing support packages, you have the opportunity to use new or updated hardware and software components such as devices, modules or library modules in the TIA Portal. Hardware components that have been added via support packages can be dragged, as usual, from the hardware catalog into the graphical view of the hardware and network editor, where they can then be configured.

In this chapter, you can find supplemental information about the support packages installed if the necessary information is not already available in the existing contents of the information system.

You can obtain support packages from the [Siemens Industry Online Support](https://support.industry.siemens.com/cs/).
