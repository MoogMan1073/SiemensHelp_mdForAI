## SINAMICS Industrial Security Configuration Manual

### Information on cybersecurity

Detailed explanations on cybersecurity in SINAMICS and Startdrive can also be found in Industry Online Support in the "[SINAMICS Industrial Security Configuration Manual](https://support.industry.siemens.com/cs/us/en/view/109810578)".

This Configuration Manual is valid as of SINAMICS FW 6.1. It contains information on the safe operation of converters in the intended operating environments, on the configuration of the security functions, and on measures to harden the system.
